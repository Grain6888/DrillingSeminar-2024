# 本システムについて
マウスやキーボードによる操作が可能な3次元CADアプリケーションです．

`lec01`から`lec03`では`C++`と`Open-GL`の基本的な使い方，`WinOGL`ではCADアプリケーション本体について扱います．

> [!note]
> + Windows11で動作することを確認しています．

> [!warning]
> + Arm版Windowsやその他のOSでは正しく動作しない可能性があります．

> [!warning]
> + ツールがグレーで表示されている場合，そのツールを使用することはできません．
> + 図形の編集は，自由図形モードおよび視点の変更モードの両方が解除されている場合にのみ有効です．
> + 多角形として閉じられていない図形（以下，追加途中の図形）に対し図形の編集を行うことはできません．
> + 追加途中の図形に対し面の表示モードを適用することはできません（ほかの閉じた図形に対し同時に適用することは可能です）．
> + 奥行きの表示モードは，視点の変更モードが選択されている場合にのみ有効です．
> + 元に戻すモードは，直前に追加した頂点を削除することのみが可能です（図形の編集により移動/削除した操作を取り消すことができるものではありません）．
> + ワイヤフレームモードは，面の表示モードが解除されており奥行きの表示モードが選択されている場合にのみ有効です．

## 2.	キャンバス
マウスを使用して図形の挿入を行うことで，挿入された図形が表示されます．また，マウスを使用して挿入した図形に対して編集を行います．

# 図形を追加する
## 1.	頂点の追加
![screenshot](https://github.com/user-attachments/assets/f7d97596-feea-41a8-85e5-1b601ec47777)

ツールバーの**自由図形追加モード**を選択後，左クリックすることで頂点を追加することができます．左クリック時に追加される頂点に対応する辺の予測が，破線で表示されます．

> [!warning]
> + カーソルが🚫の場合，その位置に頂点を追加することはできません．
> + 形状が閉じられるまで，次の形状を追加することはできません．
> + **視点の変更モード**を選択中は図形の追加操作ができません．

## 2.	頂点の追加の取り消し
ツールバーの**自由図形追加モード**を選択後，ツールバーの**元に戻す**を選択，または`Ctrl`と`Z`を同時に押下することで，頂点の追加を取り消すことができます．

> [!note]
> + 頂点の追加の取り消しは，追加した順と逆方向に実行されます．任意の点を削除する場合は，形状を閉じた後，頂点の削除を使用してください．

## 3.	キャンバスのクリア
ツールバーの**初期化**を選択することで，すべての頂点の追加を取り消すことができます．

# 図形を編集する
## 1.	頂点/辺/形状の選択
ツールバーの**自由図形追加モード**を解除後，要素を左クリックすることで選択することができます．

> [!tip]
> + `Ctrl`を押下しながら操作することで，複数の要素を同時に選択することができます．

## 2.	頂点/辺/形状の移動
要素を左クリック後，ドラッグアンドドロップすることで移動することができます．

> [!tip]
> + `Ctrl`を押下しながら操作することで，複数の要素を同時に選択することができます．

> [!warning]
> + 閉じられていない形状に対して操作を行うことはできません．

## 3.	頂点/形状の削除
要素を右クリックすることで削除することができます．
> [!warning]
> + 形状に含まれる頂点の数が3つ以下の場合，頂点を削除することはできません．形状を削除してください．
> + 閉じられていない形状に対して操作を行うことはできません．

## 4.	頂点の挿入
辺を左ダブルクリックすることで辺上に頂点を挿入することができます．

## 5.	形状の拡大・縮小
![screenshot](https://github.com/user-attachments/assets/a53d67a9-110a-4c77-9e5c-4fcfc65b82a3)

形状を左クリック後，マウスホイールを回転することで操作することができます．また，形状を左クリック後，緑色のバウンディングボックスの四隅をドラッグアンドドロップすることで操作することができます．

> [!note]
> + 拡大・縮小の中心点は，キャンバス上で赤い点として表示されます．
> + バウンディングボックスの色は，形状を左クリックすることで緑と青に順に変化します．

> [!tip]
> + キャンバス上でミドルクリックすることで中心点の位置を変更することができます．
> + `Ctrl`を押下しながら操作することで，複数の要素を同時に操作することができます．

> [!warning]
> + 閉じられていない形状に対して操作を行うことはできません．

## 6.	形状の回転
![screenshot](https://github.com/user-attachments/assets/dd93b88b-f510-4feb-b811-ff1793d32544)

形状を左クリック後，マウスホイールを回転することで操作することができます．また，形状を左クリック後，青色のバウンディングボックスの四隅をドラッグアンドドロップすることで操作することができます．

> [!note]
> + 回転の中心点は，キャンバス上で赤い点として表示されます．

> [!tip]
> + キャンバス上でミドルクリックすることで中心点の位置を変更することができます．
> + `Ctrl`を押下しながら操作することで，複数の要素を同時に操作することができます．

> [!tip]
> + バウンディングボックスの色は，形状を左クリックすることで緑と青に順に変化します．

> [!warning]
> + 閉じられていない形状に対して操作を行うことはできません．

# 表示の切り替え
## 1.	面の表示
![screenshot](https://github.com/user-attachments/assets/75bd2c66-e842-492c-9a94-2edb20c7eb49)


ツールバーの**面の表示**を選択することで，形状の面を表示することができます．

> [!warning]
> + 図形の編集中にドラッグアンドドロップをしている間は，面の表示が一時的に無効化されます．

## 2.	視点の変更
ツールバーの**視点の変更**を選択後，視点を変更することができます．

> [!note]
> + 視点を回転する場合は，キャンバス上で左ドラッグアンドドロップすることで操作することができます．
> + 視点を平行移動する場合は，`Shift`を押下しながらキャンバス上で左ドラッグアンドドロップすることで操作することができます．
> + 視点を拡大・縮小する場合は，`Ctrl`を押下しながらキャンバス上でマウスホイールを回転することで操作することができます．

## 3.	座標軸の表示
ツールバーの**座標軸の表示**を選択することで，座標軸を表示することができます．

## 4.	グリッド線の表示
ツールバーの**グリッド線の表示**を選択することで，グリッド線を表示することができます．
> [!warning]
> + 視点の変更モードが解除されている場合にのみ有効です．
> + グリッド線に沿って図形の編集をすることはできません．

## 5.	奥行きの表示
![screenshot](https://github.com/user-attachments/assets/c3f6d209-b978-40ec-ade0-073ad8f5e476)

ツールバーの**奥行きの表示**を選択することで，形状の奥行を表示することができます．また，`Shift`を押下しながらマウスホイールを回転することで奥行きの長さを調節することができます．

> [!warning]
> + 視点の変更モードが選択されている場合にのみ有効です．

> [!tip]
> + ツールバーの**面の表示**を解除することで，ワイヤフレームを表示することができます．
